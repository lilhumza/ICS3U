Single Displacement Reactions

Iron can be turned into copper!

The General Formula for a Single Displacement Reaction is

A      +     BC            B      +      AC

Note: The above reaction is for when “A” is a metal. Here are some examples:

Zn    +    2HCl         H2     +      ZnCl2

2Al     +  3CuSO4       3Cu     +    Al2(SO4)3

The General Formula II for a Single Displacement Reaction is

A      +     BC            C      +      BA

Note: The above reaction is for when “A” is a non-metal. Here are some examples:

F2    +    2NaCl        Cl2     +     2NaF

Cl2   +      2KI           I2        +     2KCl

Activity Series

Chlorine gas will react with potassium iodide to produce solid iodine and potassium chloride. However, iodine will not be able to react with potassium iodide. Bromine will not be able to react with sodium fluoride, but fluorine will react with sodium bromide. Take a close look at the halogens group.

In a single displacement reaction, the binary compound containing the halogen with a metal reacts with a halogen higher up in the periodic group. The halogen that is higher up the table will switch places with the lower halogen because the halogen that is higher is more active. Here is another example:

F2    +      2LiCl      2LiF     +     Cl2

I2      +      Ca(Cl)2     No Reactions
